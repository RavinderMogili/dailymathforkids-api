Daily Math for Kids – API

This project serves serverless API endpoints for the Daily Math site.

Quick test

/api/hello – simple health check

Endpoints

  • POST /api/register – { "nickname": "Ravi" } → { userId, nickname }
  • GET /api/status?userId=<uuid>&quizId=YYYY-MM-DD → { done, score }
  • POST /api/submit – { userId, quizId, answers: [..5..] } → { score, outOf }
